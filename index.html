Anlisis Exploratorio de Datos - Netflix Dataset

Descripcin del Proyecto

Este proyecto realiza un Anlisis Exploratorio de Datos (EDA) completo del catlogo de Netflix, utilizando las libreras ms importantes de Python para ciencia de datos:

  • Pandas: Manipulacin y limpieza de datos
  • NumPy: Operaciones numricas
  • Matplotlib: Visualizaciones bsicas
  • Seaborn: Visualizaciones estadsticas avanzadas

Objetivos

  1. Limpiar y preparar los datos
  2. Analizar patrones y tendencias del contenido
  3. Crear visualizaciones informativas
  4. Extraer insights valiosos del negocio
